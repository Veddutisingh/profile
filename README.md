## 🚀 Featured Projects

### 🤖 1. AgentForge — Autonomous AI Agent Framework
> *Agentic AI · LangChain · Python · Tool-Use · Memory*

A modular agentic AI framework where autonomous agents plan, reason, and execute multi-step tasks using tool-use, memory, and reflection loops. Designed to coordinate multiple sub-agents for complex real-world workflows.

**Key features:**
- Multi-agent coordination with role-based task delegation
- Long-term memory via vector database (Chroma)
- Real-time tool invocation: web search, code execution, file I/O
- Streaming output with progress visibility
- Pluggable LLM backends (OpenAI, Anthropic, local models)

`Python` `LangChain` `ChromaDB` `FastAPI` `React` `Docker`

---

### 🛡️ 2. VulnScope — Automated Vulnerability Assessment Platform
> *Cybersecurity · Python · Network Security · Web App Security*

A comprehensive vulnerability assessment tool that automates recon, scanning, and reporting for web applications and network infrastructure. Generates structured penetration testing reports with risk scoring.

**Key features:**
- Automated port scanning and service fingerprinting (Nmap integration)
- OWASP Top 10 vulnerability detection for web apps
- CVE database lookup for identified services
- PDF report generation with CVSS risk scores
- Dashboard UI for real-time scan progress

`Python` `Nmap` `SQLite` `Flask` `React` `Tailwind CSS`

### 🌐 3. NexaMart — Full Stack E-Commerce Platform
> *MERN Stack · JWT Auth · Stripe · Redux*

A production-grade e-commerce platform with a complete shopping experience — product catalog, cart, payments, order tracking, and an admin dashboard — built on the MERN stack with clean architecture and secure auth.

**Key features:**
- Role-based access control (Admin / Customer)
- Stripe payment integration with webhook handling
- Real-time inventory management
- JWT + HttpOnly cookie authentication
- Advanced product filtering, search, and pagination

`MongoDB` `Express.js` `React` `Node.js` `Redux Toolkit` `Stripe` `Tailwind CSS`

### 🧠 4. SentimentIQ — Real-Time Sentiment Analysis Engine
> *Machine Learning · NLP · Python · REST API*

An NLP-powered sentiment analysis system that classifies user reviews, social media posts, and customer feedback in real-time. Supports multi-class sentiment (positive / negative / neutral) with confidence scoring.

**Key features:**
- Fine-tuned BERT model achieving 94.2% accuracy on benchmark datasets
- REST API for real-time inference (< 120ms latency)
- Batch processing pipeline for large dataset analysis
- Interactive visualization dashboard with trend graphs
- Supports multilingual input (English, Hindi)

`Python` `TensorFlow` `HuggingFace` `FastAPI` `React` `Recharts`

---

### 🔗 5. ChainVault — Blockchain-Based Secure Document Notarization
> *Blockchain · Solidity · Web3.js · IPFS*

A decentralized application that allows users to notarize documents on-chain, creating tamper-proof, timestamped records stored on IPFS with hash verification on Ethereum.

**Key features:**
- Smart contract-based document hash storage on Ethereum
- IPFS integration for decentralized document storage
- MetaMask wallet authentication
- Document integrity verification with on-chain proof
- Gas-optimized Solidity contracts

`Solidity` `Ethereum` `Web3.js` `IPFS` `React` `Node.js`

---

### 📡 6. IoT SecureNet — Intrusion Detection System for IoT Networks
> *IoT Security · Machine Learning · Network Security · Python*

A real-time IDS tailored for IoT network environments. Uses ML-based anomaly detection to identify unauthorized access, DDoS patterns, and suspicious device behavior across MQTT and CoAP traffic.

**Key features:**
- Real-time packet capture and analysis
- ML-based anomaly detection (Isolation Forest + LSTM)
- MQTT broker monitoring with alert triggers
- Web dashboard for live network topology and threat map
- Logging and incident report generation

`Python` `Scikit-Learn` `TensorFlow` `MQTT` `React` `MongoDB`
